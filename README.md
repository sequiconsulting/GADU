## üìñ About

**G.A.D.U. ** is a comprehensive membership and administrative management system designed specifically for Masonic lodges. It streamlines the management of members across multiple branches (Craft, Mark, Chapter, Royal Arch Masons), tracks member progress, handles role assignments, and generates organizational reports.

Built with modern web technologies, GADU provides an intuitive interface for lodge administrators to efficiently manage their membership database while maintaining data integrity and security.
